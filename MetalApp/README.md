🚀 Metal Prices App

A React Native (Expo) mobile application that displays live prices of precious metals with a clean and premium UI.
This app simulates a real-world financial product experience with proper state handling, navigation, and structured code.

---

📱 Features

🏠 Home Screen

- Displays Gold, Silver, Platinum, Palladium
- Shows:
  - Metal name
  - Current price (₹)
  - ⏱️ Last updated timestamp
- Each metal loads independently
- Smooth card-based premium UI

---

📄 Details Screen

- Detailed view for each metal
- Shows:
  - Current price
  - Previous open price
  - Previous close price
  - Current date
  - Current time

---

⚙️ Core Functionalities

- 🔄 Independent data fetching per metal
- ⏳ Loading indicators (per item)
- ❌ Error handling with retry option
- 🔁 Pull-to-refresh support
- 🧭 Navigation using React Navigation

---

🧱 Tech Stack

- React Native (Expo)
- JavaScript (ES6+)
- React Navigation
- Custom Components Architecture

🛠️ Installation & Run

# Clone the repository
git clone <your-repo-link>

# Navigate to project
cd MetalPricesApp

# Install dependencies
npm install

# Start the app
npx expo start
